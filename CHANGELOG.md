## Versioning Scheme

This project uses Semantic Versioning:

- **MAJOR** (X.0.0): Curriculum restructuring, breaking changes to exercise interfaces
- **MINOR** (5.X.0): New exercises, additional content, new features
- **PATCH** (5.0.X): Bug fixes, documentation corrections, minor improvements

### Pre-release Tags

- `-alpha`: Early development, may be incomplete
- `-beta`: Feature complete, testing in progress
- `-rc.N`: Release candidate N, final testing
