# Jurnal de Modificări

Toate modificările notabile ale acestui proiect vor fi documentate în acest fișier.

Formatul este bazat pe [Keep a Changelog](https://keepachangelog.com/ro/1.0.0/),
și acest proiect aderă la [Versionare Semantică](https://semver.org/lang/ro/).

## [1.0.0] - 2025-01-06

### Adăugat
- Versiune inițială a WEEK4_WSLkit_RO
- Implementări protocol TEXT, BINAR și Senzor UDP
- Scripturi de configurare mediu (verify_environment.py, install_prerequisites.py)
- Configurație Docker Compose cu Portainer
- Suite de teste pentru validare exerciții
- Documentație completă în limba română
- Șabloane teme pentru acasă

### Structură
- `setup/` - Scripturi de configurare și verificare mediu
- `docker/` - Fișiere Docker Compose și Dockerfile
- `scripts/` - Scripturi de management laborator
- `src/` - Cod sursă protocoale și exerciții
- `tests/` - Suite de teste automate
- `docs/` - Documentație suplimentară
- `homework/` - Teme pentru acasă
- `pcap/` - Director pentru capturi pachete
- `artifacts/` - Director pentru fișiere generate
