## [1.0.0] - 2025-01-06

### Adăugat

- **README.md** - Ghid complet cu 7 exerciții și obiective de învățare
- **Scripturi de configurare** - verify_environment.py, install_prerequisites.py, configure_docker.py
- **Orchestrare Docker** - docker-compose.yml cu Nginx și 3 backend-uri
- **Echilibror de sarcină Python** - Suport pentru round-robin, least-conn, ip_hash
- **Client DNS** - Implementare query DNS cu afișare verbose
- **Scripturi de management** - start_lab.py, stop_lab.py, cleanup.py
- **Scripturi de captură** - capture_traffic.py pentru integrare Wireshark
- **Suite de teste** - test_environment.py, test_exercises.py, smoke_test.py
- **Documentație** - Rezumat teoretic, cheatsheet comenzi, ghid depanare
- **Teme pentru acasă** - Echilibror extins și resolver DNS cu cache

### Modificat

- Configurație Nginx optimizată pentru demonstrații educaționale
- Scripturi de curățare extinse pentru eliminare completă resurse
