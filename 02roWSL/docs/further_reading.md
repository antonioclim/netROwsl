## Cărți Recomandate

### Fundamentale

1. **Kurose, J. & Ross, K.** (2021). *Computer Networking: A Top-Down Approach* (8th ed.). Pearson.
   - Capitolul 2: Application Layer
   - Capitolul 3: Transport Layer (TCP/UDP)
   - Abordare de sus în jos, excelent pentru înțelegerea conceptelor

2. **Stevens, W. R.** (1994). *TCP/IP Illustrated, Volume 1: The Protocols*. Addison-Wesley.
   - Referință clasică pentru protocoalele TCP/IP
   - Analiză detaliată a pachetelor
   - Exemple practice cu tcpdump

3. **Tanenbaum, A. S. & Wetherall, D.** (2011). *Computer Networks* (5th ed.). Pearson.
   - Acoperire completă a rețelelor
   - Modelul OSI explicat în detaliu

### Programare Socket

4. **Rhodes, B. & Goetzen, J.** (2014). *Foundations of Python Network Programming* (3rd ed.). Apress.
   - Programare socket în Python
   - Protocoale de nivel înalt

5. **Donahoo, M. J. & Calvert, K. L.** (2009). *TCP/IP Sockets in C* (2nd ed.). Morgan Kaufmann.
   - Implementări la nivel scăzut
   - Excelent pentru înțelegerea profundă

## Subiecte Avansate pentru Studiu Independent

### Următorii Pași

1. **Programare Asincronă**
   - asyncio în Python
   - Event loops și non-blocking I/O
   - select(), poll(), epoll()

2. **Protocoale de Nivel Înalt**
   - HTTP/1.1 și HTTP/2
   - WebSockets
   - gRPC

3. **Securitate Rețele**
   - TLS/SSL
   - Autentificare și criptare
   - Atacuri comune (man-in-the-middle, etc.)

4. **Performanță și Scalabilitate**
   - Connection pooling
   - Load balancing
   - CDN-uri

5. **Rețele Moderne**
   - Software-Defined Networking (SDN)
   - Network Function Virtualization (NFV)
   - Container networking (Docker, Kubernetes)
