## Cărți Recomandate

### Nivel Introductiv

**Computer Networking: A Top-Down Approach** - Kurose & Ross
- Pornește de la aplicații și coboară spre hardware
- Bună pentru primii pași în networking
- Ediția 7 sau 8 preferată
- Capitolele relevante: 1-3

**TCP/IP Illustrated, Volume 1** - W. Richard Stevens
- Clasic absolut pentru înțelegerea protocoalelor
- Multe diagrame și exemple practice
- Referință pentru stiva TCP/IP

### Nivel Avansat

**Computer Networks** - Tanenbaum & Wetherall
- Acoperire completă de la Layer 1 la Layer 7
- Mai academic, dar solid

**Unix Network Programming** - Stevens, Fenner, Rudoff
- Biblia programării socket-urilor
- Exemple extinse în C

## Documentație Online

### RFC-uri Esențiale

| RFC | Titlu | Relevant Pentru |
|-----|-------|-----------------|
| 791 | Internet Protocol | Structura pachetelor IP |
| 793 | TCP | Stări, handshake, control flux |
| 768 | UDP | Protocol simplu, fără conexiune |
| 1918 | Private Addresses | Adrese IP private |
| 2616 | HTTP/1.1 | Protocol web (înlocuit de 7230-7235) |

Link: https://www.rfc-editor.org/

## Resurse pentru Examen

### Concepte de Revizuit

1. **Modelul TCP/IP** - cele 4 straturi și funcțiile lor
2. **Adresare IP** - notație CIDR, subnetting, adrese private
3. **TCP vs UDP** - când folosești fiecare
4. **Handshake TCP** - cei 3 pași și de ce sunt necesari
5. **Socket Programming** - bind, listen, accept, connect

### Exerciții Recomandate

- Calculează adrese de subrețea manual
- Desenează diagrama handshake-ului TCP
- Scrie un server echo TCP în Python
- Analizează o captură PCAP cu Wireshark
