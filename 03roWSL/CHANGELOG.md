## [1.0.0] - 2026-01-06

### Adăugat
- Configurație Docker Compose cu 5 servicii (server, router, client, receiver, portainer)
- Exerciții pentru broadcast UDP, multicast UDP și tunel TCP
- Scripturi de management: pornire, oprire, curățare, captură trafic
- Suite de teste pentru validarea mediului și exercițiilor
- Documentație completă în limba română
- Teme pentru acasă cu schelet de implementare
- Ghid de depanare pentru probleme frecvente
- Cheatsheet cu comenzi utile

### Infrastructură
- Imagine Docker bazată pe Ubuntu 22.04
- Instrumente de rețea: tcpdump, tshark, netcat, iperf3, scapy
- Rețea izolată 172.20.0.0/24
- Health checks pentru toate serviciile
- Profiluri Docker pentru servicii opționale

### Documentație
- README.md complet cu obiective de învățare
- Rezumat teoretic al conceptelor cheie
- Referințe RFC și bibliografice
- Rezultate așteptate pentru verificare
